# Pathfinder Aperture

> Structure at velocity. Sovereignty at the center.

Pathfinder Aperture is a GPU-accelerated cognitive telemetry architecture designed to transform high-volume data into traceable, operator-governed understanding.

The project separates structural resolution from interpretation, governance, and action. Rather than treating intelligence as a single process, Pathfinder Aperture decomposes cognition into specialized layers that operate under evidence, challenge, and governance constraints.

---

## Core Principle

Raw data is not intelligence.

Data becomes useful only when:
1. **Structure is identified** (Patterns resolved at high speed).
2. **Evidence is grounded** (Anchored in direct, high-fidelity source data).
3. **Interpretations are challenged** (Surface logical contradictions and assumptions).
4. **Governance rules are enforced** (Pre-commit compliance gatekeeping).
5. **The operator authorizes action** (Two feet on the floor, direct human consent).

Pathfinder Aperture exists to make that process explicit, tangible, and auditable.

## Cognitive Architecture Topology

```text
       [ AETHER ]         <-- Hold & Map Uncertainty
           ↓
       [ HERMES ]         <-- Document & Evidence Retrieval
           ↓
       [ RAPIDS ]         <-- GPU-Accelerated Structural Resolution
           ↓
       [ SIMON  ]         <-- Structured Multimodal Pathways & Trajectories
           ↓
       [ JEMMA  ]         <-- Critical Contradiction & Challenge Verator
           ↓
       [ DELTA  ]         <-- Substrate Grounded Cognitive Acquisition
           ↓
       [ OCTAGON]         <-- Multi-jurisdiction Policy & Sovereignty Audit
           ↓
    [ CRYSTAL BRIDGE ]    <-- Signed Operator Dispatch
```

### Layer Reference

*   **AETHER**: Maps uncertainty and identifies unknowns before premature assumptions are allowed to form.
*   **HERMES**: Retrieves evidence and grounds claims to direct target sources.
*   **RAPIDS**: Performs GPU-accelerated structural resolution. Resolves correlations, topology, statistical shifts, and high-volume patterns. *RAPIDS does not reason; RAPIDS resolves.*
*   **SIMON**: Interprets verified structure, mapping them to functional, multimodal alternatives and trajectories.
*   **JEMMA**: Challenges assumptions, exposes authority gaps, and creates productive friction.
*   **DELTA**: Measures and logs internalized learning, turning raw observations into acquired permanent directives.
*   **OCTAGON**: Governs execution against strict compliance, verification levels, and operator protections.
*   **CRYSTAL BRIDGE**: The final controlled output projector. No directive leaves the node without explicit operator verification and digital signature.

*   **Fast Path Processing**: NVIDIA RAPIDS, cuDF, cuML, cuGraph (GPU-accelerated statistical mapping/acceleration).

*   **Sovereign Visualization Core**: Vite, React 18, Tailwind CSS, Lucide Icons, and Motion transitions.
*   **Persistence & Ledger Subsystem**: Firebase Firestore (Substrate Delta Rule Repository and Audit Logging).

## Getting Started

### Prerequisites

*   **Node.js** (v18 or higher)
*   **NVIDIA CUDA Environment** (For local GPU-accelerated fast paths, optional fallback provided)
*   **Gemini API Key** (Configured as `GEMINI_API_KEY` for real-time grounded research telemetry)

### Installation & Run

1.  Clone the repository.
2.  Install all dependencies:
    ```bash
    npm install
    ```
3.  Set up environment parameters in `.env`:
    ```env
    GEMINI_API_KEY=your_gemini_api_key_here
    ```
4.  Launch the development cockpit:
    ```bash
    npm run dev
    ```
