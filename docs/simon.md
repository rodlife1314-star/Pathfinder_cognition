# SIMON Trajectory Engine Spec

## Translating Structure into Alternatives

While the RAPIDS layer provides topological structure, it does not interpret system implications. That translation is the responsibility of the **SIMON Trajectory Engine**. 

SIMON maps the compressed telemetry representations into finite, actionable paths called **Postures** or **Trajectories**.

---

## 1. Selected Trajectory Profiles

Pathfinder Aperture structures decision paths into four standardized navigational postures:

| Posture Type | Core Rule Anchor | Objective | Target Action Bounds |
| :--- | :--- | :--- | :--- |
| **Conservative** | Governance 1.1 | Capital Preservation | Minimize exposure, seek safe liquidity havens |
| **Moderate** | Basis Balance 3.2 | Convergence Capture | Exploit stable basis anomalies, hedge delta |
| **Aggressive** | Thrust Rule 5.9 | Basis Expansion | Capture high-momentum breakouts, unhedged spread |
| **Standby** | Sovereign Core 1.01 | Inaction / Calibration | Maintain absolute neutrality, analyze uncertainty |

---

## 2. Dynamic Option Formulation

SIMON coordinates with the HERMES evidence pipeline on launch to analyze matching guidelines. The algorithm structures a trajectory by compiling:
1.  **Pathway Title**: e.g., "Basis convergence arbitrage".
2.  **Governing Rule Anchor**: e.g., "Governance 3.22: Dynamic high-velocity hedge triggers must maintain direct Coinbase spot reconciliation".
3.  **Traceable Claims**: A series of physical assertions required to justify this posture.
4.  **Baseline Target Parameters**: Target price convergence, speed of execution, and required liquidity thresholds.

---

## 3. Operational Interface Layout

The operator views the SIMON alternatives within the **Sovereign Observer Aperture** as a selectable ledger of trajectories:

```text
  [ SIMON PROTOCOL REGISTRY ]
  ┌─────────────────────────────────────────────────────────────┐
  │ ◉ TRAJECTORY: BASIS CONVERGENCE ARBITRAGE (MODERATE)        │
  │   Rule: "Basis stability triggers require Coinbase Spot"     │
  │   Status: Reconciled (+140 points)                          │
  ├─────────────────────────────────────────────────────────────┤
  │ ◯ TRAJECTORY: CONSERVATIVE COLLATERAL RECOVERY (CONSERVATE)  │
  │   Rule: "Collateral recovery triggers only on standby rules"│
  │   Status: Grounded (+85 points)                             │
  └─────────────────────────────────────────────────────────────┘
```

Selecting a trajectory shifts the entire observer field:
*   **AETHER** adjusts its tension markers.
*   **HERMES** highlights matching evidence quotes.
*   **JEMMA** runs a friction compile on the target claims.
*   **OCTAGON** audits compliance levels.
*   **CRYSTAL BRIDGE** loads the final verified action payload.
